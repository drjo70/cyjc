## 🔧 문제 해결 가이드

### ❌ 자주 발생하는 문제

#### 1. DB 연결 실패
**증상**: "Database connection failed" 오류
**해결방법**: 
- `config/database.php`의 DB 정보 확인
- Cafe24 호스팅 DB 상태 확인
- `test_db.php`로 연결 테스트

#### 2. 한글 깨짐
**증상**: 인물명이 "???" 또는 깨진 문자로 표시
**해결방법**:
- DB 문자셋이 `UTF8MB4`인지 확인
- `family_members` 테이블 문자셋 확인
- 웹서버 PHP 설정에서 `default_charset = UTF-8` 확인

#### 3. PWA 설치 안됨
**증상**: "홈 화면에 추가" 알림이 안 뜸
**해결방법**:
- HTTPS 프로토콜 사용 확인 (HTTP는 PWA 지원 안됨)
- `manifest.json` 파일 접근 가능 확인
- PWA 아이콘 파일들 존재 확인 (`assets/` 폴더)

#### 4. 관리자 로그인 안됨
**증상**: "비밀번호가 틀렸습니다" 메시지
**해결방법**:
- 정확한 비밀번호 입력: `cyjc2024admin`
- 브라우저 쿠키 삭제 후 재시도
- 세션 폴더 권한 확인 (웹서버)
